## Overview

This module integrates **Prometheus** metrics with your **Express.js** application to monitor performance. It tracks default system metrics like memory usage, custom metrics such as request durations, and the total number of HTTP requests. This setup allows you to easily expose application metrics that can be scraped by Prometheus for performance monitoring.
